First, you should load recharts:

library(recharts)

1 Introduction

Scatter plot includes: - simple scatter/point - bubble

For scatter plots, x and y must both be numeric.

2 Simple Scatter Plot

2.1 Single Series

echartr(iris, Sepal.Width, Petal.Width)

It is equivalent to

echartr(iris, x=Sepal.Width, y=Petal.Width, type='scatter')
echartr(iris, Sepal.Width, Petal.Width, type='point')
echartr(iris, Sepal.Width, Petal.Width, type='bubble')

2.2 Multiple Series

echartr(iris, Sepal.Width, Petal.Width, Species)

3 Bubble Chart

The key is to pass weight variable. If weight is accepted, no matter the type variable is ‘scatter’, ‘point’ or ‘bubble’, a bubble chart will display.

echartr(iris, Sepal.Width, Petal.Width, weight=Petal.Length)

4 Scatter Plot with Timeline

Timeline calls for numeric or time variable. When a character variable is passed in, echarts coerces it to numeric.

echartr(iris, Sepal.Width, Petal.Width, z=Species)

5 Aesthetics

Of course you can apply setTheme or setSymbols to modify the charts

g <- echartr(iris, Sepal.Width, Petal.Width, Species)
  • Apply ‘vintage’ theme and set the width 500 px, height 350 px.
g %>% setTheme('vintage', width=500, height=350)
  • Apply non-standard symbols c(‘star6’, ‘emptyCircle’, ‘heart’) to symbolList.
rm(g)
g = echartr(iris, Sepal.Width, Petal.Width, Species) %>% 
  setTheme('macarons') %>% setSymbols(c('star6', 'emptyCircle', 'heart')) 
g
  • Move legend to 12 o’clock position, toolbox to 9 o’clock position.
  • Set title = ‘Iris Flowers’ at the 6 o’clock position.
rm(g)
g = echartr(iris, Sepal.Width, Petal.Width, Species) %>% 
  setLegend(pos=12) %>% setToolbox(pos=9) %>% 
  setTitle("Iris Flowers", pos=6) %>% setTheme('helianthus')
g

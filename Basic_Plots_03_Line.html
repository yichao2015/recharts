First, you should load recharts:

library(recharts)

1 Introduction

Line plot includes 8 basic types:

  • line: line
  • stacked line: line_stack|stacked_line
  • smooth line: line_smooth|smooth_line|curve
  • smooth stacked line: line_smooth_stack|line_stack_smooth|stacked_smooth_line|curve_stack|stacked_curve
  • area: area
  • stacked area: area_stack|stacked_area
  • smooth area: area_smooth|smooth_area
  • stacked smooth area: area_smooth_stack|area_stack_smooth|stacked_smooth_area

The keys are:

  • character x and numeric y
  • each combination level of x (and series) can only have one y data point
  • Tiled and stacked bar/column charts can shift from one to another by clicking type shift buttons in the toolbox widget.
  • Line/area can shift to bar charts easily using buttons in toolbox widget.

2 Function Call

echartr(data, x, y, <series>, <z>, <type>)
Arg Requirement

data

source data in the form of data.frame

x

character independent variable. Other type will be coerced to factors. Only the first one is accepted if multiple variables are provided.

y

numeric dependent variable. Only the first one is accepted if multiple variables are provided.

series

data series variable which will be coerced to factors. Only the first one is accepted if multiple variables are provided.

z

timeline variable which will be coerced to factors. Only the first one is accepted if multiple variables are provided.

type

‘line’, ‘line_stack’, ‘stacked_line’, ‘line_smooth’, ‘smooth_line’, ‘curve’, ‘line_smooth_stack’, ‘line_stack_smooth’, ‘stacked_smooth_line’, ‘curve_stack’, ‘stacked_curve’,‘area’, ‘area_stack’, ‘stacked_area’, ‘area_smooth’, ‘smooth_area’, ‘area_smooth_stack’, ‘area_stack_smooth’, ‘stacked_smooth_area’

3.1 Data Preparation

Let’s look into airquality dataset embeded in the package datasets. The structure of the data.frame is as below:

aq <- airquality
aq$Date <- as.Date(paste('1973', aq$Month, aq$Day, sep='-'))
aq$Day <- as.character(aq$Day)
aq$Month <- factor(aq$Month, labels=c("May", "Jun", "Jul", "Aug", "Sep"))

3.2 Line Chart

3.2.1 Singular Series

type is set ‘line’.

echartr(aq, Date, Temp, type='line') %>%
    setTitle('NY Temperature May - Sep 1973')

3.2.2 Multiple Series

echartr(aq, Day, Temp, Month, type='line') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.2.3 Line Chart with Timeline

echartr(aq, Day, Temp, z=Month, type='line') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.3 Stacked Line Chart

Set type ‘line_stack’ or ‘stacked_line’.

echartr(aq, Day, Temp, Month, type='line_stack') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.4 Smooth Line Chart

Set type ‘curve’, ‘line_smooth’ or ‘smooth_line’.

echartr(aq, Day, Temp, Month, type='curve') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.5 Stacked Smooth Line Chart

Set type ‘line_smooth_stack’, ‘line_stack_smooth’, ‘stacked_smooth_line’, ‘curve_stack’ or ‘stacked_curve’.

echartr(aq, Day, Temp, Month, type='curve_stack') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.6 Area Chart

Area chart is exactly the same with line chart except that it has areaStyle feature specialized.

Set type ‘area’.

echartr(aq, Date, Temp, type='area') %>%
    setTitle('NY Temperature May - Sep 1973') %>% 
    setSymbols('emptycircle')

3.7 Stacked Area Chart

Set type ‘area_stack’ or ‘stacked_area’.

echartr(aq, Day, Temp, Month, type='area_stack') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')

3.8 Smooth Area Chart

Set type ‘area_smooth’ or ‘smooth_area’.

echartr(aq, Date, Temp, type='area_smooth') %>%
    setTitle('NY Temperature May - Sep 1973') %>% 
    setSymbols('emptycircle')

3.9 Stacked Smooth Area Chart

Set type ‘area_smooth_stack’, ‘area_stack_smooth’ or ‘stacked_smooth_area’.

echartr(aq, Day, Temp, Month, type='area_smooth_stack') %>%
    setTitle('NY Temperature May - Sep 1973, by Month') %>% 
    setSymbols('emptycircle')
